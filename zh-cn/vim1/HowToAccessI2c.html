如何使用I2C接口

这篇文档介绍如何在Android和Ubuntu下使用I2C。

前提条件

  • Ubuntu V180531或更新版本
  • Android V180619或更新版本

连线

I2C A
Sensor SCL <-> I2C_SCK_A (PIN 22)
Sensor SDA <-> I2C_SDA_A (PIN 23)
Sensor GND <-> GND
Sensor VCC <-> 5V或3.3V //取决于传感器供电

I2C B
Sensor SCL <-> I2C_SCK_B (PIN 25)
Sensor SDA <-> I2C_SDA_B (PIN 26)
Sensor GND <-> GND
Sensor VCC <-> 5V或3.3V //取决于传感器供电
