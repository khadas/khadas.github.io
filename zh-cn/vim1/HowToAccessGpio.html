如何访问GPIO

这篇文档介绍如何在Android和Ubuntu下访问GPIO。

准备工作

固件版本必须满足以下条件：

  • Android M >= V170603
  • Android N >= V170421
  • Ubuntu Server >= V180712
  • Ubuntu Mate >= V180531

如何获取GPIO数值

你可以通过GPIO banks和pins来获取。不同版本的内核数值不同。

Number(GPIOH_4) = bank + pin = 155 - 10 + 30= 175
Number(GPIOAO_6) = bank + pin = 145 - 0 + 6 = 151

Pins:
