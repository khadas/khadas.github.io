Write SD Image to eMMC

This guide will show you how to write an SD/USB firmware to the eMMC.

Boot from SD/USB

  1. Write the firmware to an SD card or USB thumbdrive. To make a bootable media, please refer to: Install Image to SD-USB Storage.
  2. Boot the system from SD card or USB thumbdrive. Please refer to the boot method: Boot From External Media.

Write OS Image to eMMC

  1. Run the eMMC installation script:
  1. After the script starts, select option 1 to install the system into your eMMC.
  1. Erase the eMMC.
  1. Format the eMMC, and burn the OS image.
  1. After the installation is complete, select Power off. Remove the SD card or thumbdrive, then reboot from eMMC.

If you don’t want to boot from the eMMC immediately, you can choose Exit.
